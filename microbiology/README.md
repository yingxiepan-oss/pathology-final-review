# 医学微生物学C最后35小时冲刺站

直接双击 `index.html` 即可离线使用。若浏览器限制本地文件脚本，可在本目录启动任意静态文件服务器后访问。

## 建议使用顺序

1. 英文术语速认。
2. 36个必背名词：先12个跨年高频，再13个*号新增，最后11个历年单次。
3. 8个高频简答背得分骨架。
4. 一本通11个病例。
5. 39道可靠判分选择题。
6. 错题／需回看和最后30分钟。

## 数据范围

- 名词解释：8份卷共41次可辨认出现，归并为23个母题；名词资料有31个带*项目，其中18个与历年题重叠、13个为新增，最终必背并集36个；另有2题原回忆缺失。
- 简答题：15个母题，其中8个跨年复现。
- 病例题：一本通PDF p40-46的11题全部整理。
- 选择题：39道答案可可靠判定的历年题或同源训练。
- 历年题：8份回忆、184个可辨认项目。
- 原文待核对：13项，不进入统计分母和判分训练。

所有学习状态仅保存在当前浏览器。更换浏览器或清理网站数据后，进度不会自动迁移。
