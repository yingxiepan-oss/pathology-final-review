# 医学微生物学C最后35小时冲刺站

直接双击 `index.html` 即可离线使用。若浏览器限制本地文件脚本，可在本目录启动任意静态文件服务器后访问。

## 建议使用顺序

1. 英文术语速认：看英文并手动输入中文名称。
2. 36个必背名词：先12个跨年高频，再13个*号新增，最后11个历年单次。
3. 8个高频简答背得分骨架。
4. 一本通11个病例。
5. 39道可靠判分选择题。
6. 错题／需回看和最后30分钟。

## 数据范围

- 名词解释：8份卷共41次可辨认出现，归并为23个母题；名词资料有31个带*项目，其中18个与历年题重叠、13个为新增，最终必背并集36个；另有2题原回忆缺失。
- 简答题：15个母题，其中8个跨年复现。
- 病例题：一本通PDF p40-46的11题全部整理。
- 选择题：39道答案可可靠判定的历年题或同源训练。
- 历年题：8份回忆、184个可辨认项目。
- 原文待核对：13项，不进入统计分母和判分训练。

所有学习状态仅保存在当前浏览器。更换浏览器或清理网站数据后，进度不会自动迁移。

选择题支持反复刷题：可逐题“重做”，也可一键开始新一轮；新一轮仅清空选择作答记录，保留错题、需回看和已掌握状态。

名词解释默认显示“3分考场版”，简答题默认显示“4分考场版”；原完整答案保留在“完整理解版”折叠区，不要求逐字背诵。

病例分析保留一本通PDF p40-46的完整原题、全部病例数据、小问和选择项，病例答案不套用名解/简答精简规则。

其中8道英文病例提供完整中文翻译，可逐题或统一显示／隐藏；英文原题始终保留。
